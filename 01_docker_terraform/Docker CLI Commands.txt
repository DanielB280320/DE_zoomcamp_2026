Docker CLI:

Postgre with docker: Create and run PostgreSql server 
	- docker run -it --rm `						
  	  -e POSTGRES_USER="root" `					
 	  -e POSTGRES_PASSWORD="root" `				
	  -e POSTGRES_DB="ny_taxi" `				
	  -v ny_taxi_postgres_data:/var/lib/postgresql ` 
	  -p 5432:5432 `							
	  --network=pg-network `					
  	  --name pgdatabase `						
	  postgres:18								

	- uv add --dev pgcli						
	- uv run pgcli -h localhost -p 5432 -u root -d ny_taxi 		

Execute an ingestion script from docker container using parameters defined in the CLI
	- docker run -it --rm --network=pg-network taxi_ingest:v001 `
  	  --pg-user=root `
	  --pg-password=root `
	  --pg-host=pgdatabase `
	  --pg-port=5432 `
	  --pg-db=ny_taxi `
	  --target-table=yellow_taxi_trips_2021_01 `
	  --year=2021 `
	  --month=1 `
 	  --chunksize=100000

	Execute a pgadmin image: 
	- docker run -it `
	  -e PGADMIN_DEFAULT_EMAIL="admin@admin.com" `
      -e PGADMIN_DEFAULT_PASSWORD="root" `
 	  -v pgadmin_data:/var/lib/pgadmin `
 	  -p 8085:80 `
 	  --network=pg-network `
 	  --name pgadmin `
 	  dpage/pgadmin4
